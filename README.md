# 🌦️ Smart Crop Protection Perimeter Alarm

## 🏆 Hackathon Project

A Smart Weather Monitoring System is an IoT-based solution that collects and monitors environmental parameters such as temperature, humidity, atmospheric pressure, rainfall, and light intensity in real time. The system provides weather insights through a cloud dashboard and helps users make informed decisions in agriculture, environmental monitoring, and smart city applications.

---

## 📌 Problem Statement

Traditional weather monitoring systems are often expensive and inaccessible for small-scale users. This project provides an affordable, real-time, and remote weather monitoring solution using IoT technology.

---

## 🎯 Objectives

- Monitor weather conditions in real time.
- Display environmental data locally and remotely.
- Store weather data on a cloud platform.
- Generate alerts during abnormal weather conditions.
- Provide an affordable and scalable weather monitoring solution.

---

## ⚙️ Working Principle

1. Sensors continuously monitor weather parameters.
2. ESP32 collects and processes sensor data.
3. Weather data is displayed on an OLED/LCD display.
4. Data is uploaded to the cloud via Wi-Fi.
5. Users can view weather information through a web dashboard or mobile app.
6. Alerts are generated when environmental conditions exceed predefined thresholds.
